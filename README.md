**42 Mastery Suite** is a comprehensive, gamified educational platform designed to master the core concepts of the "42" curriculum. It features **14 distinct game modes** covering everything from algorithmic logic to network topology.

### Core Modules
*   **Algorithmic Path:** Visual programming & recursion.
*   **NetConfig 42:** IP addressing, subnetting, and routing simulation.
*   **DataStruct Duel:** Time complexity and data structure optimization.
*   **ShellScript Scramble:** Bash terminal mastery.

### Advanced Modules
*   **Graph Theory Quest:** BFS, DFS, Dijkstra visualization.
*   **Dynamic Grid Planner:** Dynamic programming optimization.
*   **Number Theory Cipher:** Cryptography & Modular arithmetic.
*   **Sorting Wars:** Algorithm visualization & efficiency.

### Concurrency & Logic
*   **Deadlock Manager:** Thread synchronization.
*   **Pipeline Flow:** Asynchronous data processing.
*   **Logic Puzzles:** Zebra Grid, Boolean Circuits, Tangrams, N-Puzzle.
